# Get Videos Data

This project will get videos content data and display in a decent grid list.
The videos list contains video thumbnail image, title and artist of video.
To check out this project, lets start by cloning the repository:

```
git clone https://github.com/hemantsain/xitetv.git
cd xitetv
yarn
npx pod-install
```

## How to run the App

- iOS
  - run command 
  ```
  yarn ios 
  or
  npx react-native run-ios
  ```

- Android
  - run command 
  ```
  yarn android
  or
  npx react-native run-android
  ```
